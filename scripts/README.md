# Helper Scripts

This folder contains optional helper scripts used during development. These are not required for normal builds.

- commit-batches.sh, commit-simple.sh, fix-and-commit.sh, quick-commit.sh: Convenience Git helpers
- restore-files.sh: Restore local files from backups
- build.js, implement-features.cjs: Experimental build/feature scaffolding
- create-audio.js: Audio asset generation helpers
- test-story-scroll.js: Experimental test runner

Use with caution and prefer standard npm scripts in package.json where possible.
