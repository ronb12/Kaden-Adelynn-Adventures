# Kaden & Adelynn Space Adventures

A space shooter game built with React and Vite.

## Features

- Space shooter gameplay with multiple waves and levels
- Character and ship selection
- Power-ups and upgrades
- Score tracking with shooting accuracy percentage
- Boss battles
- High score leaderboard (local and cloud)
- Responsive design for mobile and desktop

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Deploy

The app is deployed to Firebase Hosting:

```bash
firebase deploy --only hosting
```

## Game Features

- **Shooting Accuracy Tracking**: Real-time accuracy percentage displayed in the scoreboard
- **Multiple Weapons**: Laser, spread, plasma, missiles, and more
- **Power-ups**: Shields, rapid fire, slow motion, coin doubler
- **Boss Fights**: Epic boss battles at the end of each wave
- **Achievements**: Unlock achievements as you play
- **Store**: Purchase upgrades with coins earned in-game

## Technologies

- React
- Vite
- Canvas API for game rendering
- Firebase for hosting and cloud scores
