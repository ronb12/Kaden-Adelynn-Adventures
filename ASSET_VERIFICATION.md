#### How to Verify:
1. Check if you purchased from an asset store:
   - Unity Asset Store
   - Itch.io
   - Kenney.nl
   - Other asset marketplaces

2. Check if downloaded from free sources:
   - OpenGameArt.org
   - Freesound.org
   - Other free asset sites

3. Check if created by you:
   - Blender scripts
   - Procedurally generated
   - Hand-drawn

4. Check file metadata:
   ```bash
   # Check for embedded metadata
   exiftool Spaceship_0.png
   # Or
   mdls Spaceship_0.png
   ```
