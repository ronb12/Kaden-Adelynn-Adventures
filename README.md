# 🏛️ Kaden & Adelynn Adventures - Temple Run! 🏃‍♂️

A fun and simple Temple Run-style endless runner game for kids! Run through ancient temple ruins, collect coins, and avoid obstacles.

## 🎮 **GAME STATUS: FULLY WORKING!** ✅

The game is now a **single HTML file** that works immediately in any web browser!

## 🚀 **How to Play**

### Quick Start
1. **Open `index.html`** in any web browser
2. **Click "🏃‍♂️ START GAME!"** to begin
3. **Use the controls** to run, jump, and collect coins!

### Controls
**Desktop/Keyboard:**
- **SPACE** or **↑** - Jump over obstacles
- **↓** - Slide under obstacles  
- **←** - Move left
- **→** - Move right

**Mobile/Touch:**
- **⬆️** - Jump
- **⬇️** - Slide
- **←** - Move left
- **→** - Move right
