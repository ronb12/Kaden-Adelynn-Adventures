# 🚀 Kaden & Adelynn Adventures - React Game

A fun space shooter game built with ReactJS, designed for both mobile and web platforms!

## 🎮 Play the Game

**Live Demo**: https://kaden---adelynn-adventures.web.app

## 🎯 Game Features

- **Space Shooter Action**: Control your spaceship and defeat enemy ships
- **Keyboard Controls**: Use arrow keys or WASD to move, Spacebar to shoot
- **Progressive Difficulty**: Game gets more challenging as you advance
- **Responsive Design**: Works on both desktop and mobile devices
- **Pause Functionality**: Pause and resume the game anytime

## 🎮 How to Play

1. **Start Game**: Click "Start Adventure" from the main menu
2. **Move**: Use arrow keys or WASD to control your spaceship
3. **Shoot**: Hold Spacebar to fire at enemies
4. **Survive**: Avoid enemy ships and collect points
5. **Pause**: Click the pause button (⏸️) to pause the game

## 🏗️ Technical Details

- **Built with**: React 18, Canvas API
- **Deployed on**: Firebase Hosting
- **Source Code**: Available on GitHub
- **Responsive**: Mobile-first design approach

## 🚀 Local Development

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm start
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

4. **Deploy to Firebase**:
   ```bash
   npm run deploy-firebase
   ```
