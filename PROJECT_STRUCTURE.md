## 🔧 File Purposes

### Core Files
- **`index.html`** - Main game interface with PWA support
- **`manifest.json`** - Progressive Web App configuration
- **`README.md`** - Comprehensive project documentation

### Styling
- **`css/style.css`** - Complete responsive design and animations

### Game Logic
- **`js/main.js`** - Game engine, rendering, and mechanics
- **`js/sw.js`** - Service worker for offline functionality

### Assets
- **`assets/images/`** - All game graphics and sprites
- **`assets/sounds/`** - Prepared for future audio implementation
