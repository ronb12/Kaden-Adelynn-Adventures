# App Store Assets Checklist

This checklist helps you prepare all required and recommended assets for App Store submission.

## Required Assets

- [ ] App Icon (1024x1024 PNG, no transparency)
- [ ] iPhone 6.7" screenshot (1284x2778)
- [ ] iPhone 6.5" screenshot (1242x2688)
- [ ] iPhone 5.5" screenshot (1242x2208)
- [ ] iPad Pro 6th Gen 12.9" screenshot (2048x2732)
- [ ] iPad Pro 2nd Gen 12.9" screenshot (2048x2732)
- [ ] iPad Pro 11" screenshot (1668x2388)
- [ ] App Preview video (optional, 15-30s, .mov)
- [ ] Promotional text (max 170 chars)
- [ ] Description (max 4000 chars)
- [ ] Keywords (max 100 chars, comma-separated)
- [ ] Support URL
- [ ] Marketing URL (optional)
- [ ] Privacy Policy URL

## Recommended Assets

- [ ] Feature graphic/banner (for marketing)
- [ ] Additional screenshots (showing gameplay, menus, achievements, etc.)
- [ ] Localized screenshots and descriptions
- [ ] Game Center/leaderboard screenshots
- [ ] Accessibility feature screenshots

## Tips
- Use the Xcode Simulator or real devices to capture screenshots at required resolutions.
- Use the blender/create_app_icon.py script for icon generation.
- Store all assets in ios/AppStoreAssets/ for easy access.
- Double-check for device frame overlays and text legibility.
