# 🚀 Kaden & Adelynn Space Adventures

An epic space shooter game built with React, featuring triangle ships, power-ups, and exciting space combat!

## 🌟 Features

- **25 Lives System**: Players start with 25 lives for extended gameplay
- **Rapid Fire Power-up**: Collect the ⚡ power-up for super-fast shooting
- **Multiple Power-ups**: Health, power, score, and rapid fire enhancements
- **Dynamic Enemy AI**: Enemies that shoot back and move in patterns
- **Beautiful Starfield**: Animated background with moving stars
- **Professional UI**: Clean, modern interface with health bars and statistics
- **High Score System**: Persistent high score tracking
- **Responsive Controls**: WASD or arrow keys for movement, spacebar for shooting

## 🎮 How to Play

1. **Movement**: Use WASD or Arrow Keys to navigate your spaceship
2. **Shooting**: Press SPACEBAR to fire weapons
3. **Power-ups**: Collect emojis for various enhancements:
   - 💎 Score boost (+500 points)
   - ⭐ Power upgrade (increases weapon spread)
   - ❤️ Health restoration (+40 health)
   - ⚡ Rapid Fire (3 seconds of super-fast shooting)
   - 🛡️ Shield protection
4. **Objective**: Destroy enemy ships while avoiding their bullets
5. **Survival**: You have 25 lives to complete your mission!

## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd kaden-adelynn-space-adventures
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
```

## 🎯 Game Controls

## 🏆 Scoring System

- **Enemy Destroyed**: +100 points

- **Accuracy**: Based on shots fired vs. hits

## 🛠️ Technical Details

- **Framework**: React 18
- **Graphics**: HTML5 Canvas
- **State Management**: React Hooks
- **Styling**: CSS-in-JS with inline styles
- **Build Tool**: Create React App

## 🎨 Game Assets

- **Player Ship**: Blue triangle with white outline
- **Enemy Ships**: Red triangles
- **Bullets**: Cyan player bullets, red enemy bullets
- **Power-ups**: Emoji-based visual indicators
- **Background**: Animated starfield with parallax effect
